# Strapi Skills

Strapi is preparing a set of **public skills** to help you build Strapi applications with AI coding assistants.

These skills will provide Strapi-specific guidance for common development workflows—content modeling, APIs, plugins, configuration, and more—so assistants can follow official Strapi patterns and best practices.

## Status

**Coming soon.** This repository is not ready for use yet. Check back here for updates as skills are published.

## What are skills?

Skills are reusable instruction files that coding agents (such as Cursor) can load for domain-specific tasks. Strapi skills will encode Strapi 5 conventions, CLI usage, and documentation-first workflows to improve the quality and consistency of generated code.
